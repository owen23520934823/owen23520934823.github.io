# Healthcare Campaign — Static Site

This is a static, responsive advocacy website built with HTML, CSS, and JavaScript. It is designed to be accessible, mobile-friendly, and to communicate a humanitarian campaign focused on supporting community-based healthcare for women in Afghanistan.

Files:
- `index.html` — main page with all sections.
- `css/styles.css` — styling, responsive layout, and animations.
- `js/script.js` — smooth scroll, fade-in effects, and animated counters.

How to view:
1. Open `C:\Users\Dell\Desktop\msf website\index.html` in any modern browser.
2. No build step required.

Notes and accessibility:
- Uses semantic HTML and high-contrast color palette.
- Buttons and links are keyboard accessible.
- Images use `alt` attributes and `loading="lazy"` where appropriate.
